# 🎨 Installation du thème Azuriom

## 📄 Informations préliminaires

> Pour activer la page "Nous rejoindre", créez une page dans le panel **Azuriom** avec l’URL (slug) suivante :  
> `nous-rejoindre`

---

## 1. Installation du thème

### ✅ Pour les utilisateurs familiers avec GitHub

1. Clonez le dépôt dans le dossier des thèmes de votre site Azuriom :  
   `resources/themes`

---

### 🚫 Pour les utilisateurs **non familiers** avec GitHub

1. Accédez au dépôt du thème.
2. Cliquez sur le bouton vert **"Code"** (`<> Code`) en haut à droite.
3. Sélectionnez **"Download ZIP"**.
4. Décompressez l’archive téléchargée dans :  
   `resources/themes`
---

## 2. Détection du thème dans Azuriom

1. Renommez le dossier du thème pour qu’il corresponde à l’**id** défini dans son fichier `theme.json`.
2. Accédez à votre panel Azuriom et activez le thème.

**Exemple :**

Contenu de `resources/themes/serveurliste/theme.json` :

```json
{
  "id": "serveurliste",
  "name": "ServeurListe",
  "description": "Theme for ServeurListe",
  "version": "1.0.0",
  "url": "https://www.serveurliste.com",
  "azuriom_api": "1.0.0",
  "authors": [ "[Dixept.fr] Bricec6" ]
}
```
