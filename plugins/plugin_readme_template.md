**Achievements** is an Azuriom plugin that allows you to create goals for your users using connectors from existing plugins such as **Vote, Shop, Forum, Suggest, Review, Achievement**.

You can create a **connector**, define what triggers the goal, and configure the rewards associated with completing it.

**Example:**
- **Name**: Vote 100 times for the server
- **Hook**: Vote
- **Trigger**: vote amount
- **Amount**: 100
- **Start day (for count)**: Today

Once a user completes a goal, they can receive rewards configured for that objective:
- Shop points from the **Shop** plugin
- One or more server commands
- A set number of trophies

The plugin also includes global settings:
- Customize the name used for trophies (e.g., “Trophy”)
- Change the icon or use a custom image
- Leaderboard showing players with the most trophies

## Features :

#### 🏆 Achievements Management
- Create and manage goals for users
- Define connectors and triggers (Vote, Shop, Forum, Suggest, Review, Achievement)
- Set the amount required to complete a goal
- Configure rewards: currency, server commands, trophies
- Global settings: customize trophy name and icon
- Visibility per role

---

#### 📊 Player Rankings
- Leaderboard of players with the most trophies
- Visualize progress per user
- Custom leaderboard title

---

#### 🔧 Rewards Customization
- Configure rewards per goal
- Assign multiple server commands
- Reward in Shop plugin currency
- Trophies
- Reset per player
- Reset all

---

#### 🔐 Permissions
- Control who can create and manage goals

### 🔗 Hooks & Triggers

Below is the full list of available hooks and triggers used to connect Achievements with other plugins.

| **Hook**       | **Trigger**              |
|----------------|--------------------------|
| **vote**       | `voted`, `leaderboard`   |
| **shop**       | `spent`, `points_spent`, `item_purchase` |
| **suggest**    | `post`, `like_received`  |
| **review**     | `post`                   |
| **forum**      | `message`, `topic`, `like_received` |
| **post**       | `like`                   |
| **achievement**| `achievement_trophy_amount` |
