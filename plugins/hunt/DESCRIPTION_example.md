**Scratch Game** is an Azuriom plugin that adds a full scratch-card system to your website.

Players can buy scratch cards with site currency, reveal rewards with an interactive scratch canvas, and claim money and/or server command rewards.

### Features

#### Scratch Cards Management

- Create, edit, enable/disable, and delete scratch cards.
- Create a free game every X minutes
- Configure card price in site currency.
- Set cover image (before scratch) and background image (after scratch).
- Link one card to multiple rewards.
- Restrict card visibility/playability by Azuriom role.
- Public cards supported (no role required).

#### Rewards Management

- Create, edit, enable/disable, and delete rewards.
- Configure weighted chance (`%`) per reward.
- Money reward support (site currency).
- Server commands reward support (single or multiple commands).
- Optional reward image.
- Multi-server assignment for command execution.
- `need_online` mode supported for AzLink servers.
- Validation safeguards:
    - at least one reward type required (money or commands),
    - commands require at least one server,
    - `need_online` requires compatible AzLink setup.

#### Player Experience

- Public scratch catalog page.
- Authentication required for buying/claiming.
- Buy-and-play flow with balance check.
- Interactive scratch canvas reveal page.
- Pending tickets section on home page.
- User history page with status (pending/claimed), gains, and command details.
- Replay/continue flow for unfinished tickets.

#### Logs and Reliability

- Complete play logs (card, user, reward, price paid, money given, commands executed, metadata).
- Claim flow protected against duplicate reward dispatch.
- Transaction-based purchase/claim flow.
- Admin logs page with filters (user/card/search).

#### Admin & Security

- Plugin navigation in Azuriom admin dashboard.
- Logs for auditability.

### Reward placeholders

You can use these placeholders in reward commands:

- `{player}`
- `{user}`
- `{scratch}`
- `{card}`
- `{reward}`
- `{steam_id}`
- `{steam_id_32}`
